How to play Jotto

 Jotto is a word guessing game. The goal is to guess another player's secret 5-letter word. After each guess, if the correct word was not guessed, the other player responds with the number of letters in the guess that match the secret word. The position does not matter, but duplicates do count. For example:
Secret word Guess word Matching letters
boost defer 0
raise roast 3
feeds freed 4
rated tread 5

Using the program

 The program will try to guess the user's secret word. Start by loading a list of 5 letter words. One is provided in folder "jotto/resources/" Or a custom list can be loaded. The words are processed, and a Jotto file is built. You can save this file for faster loading of words next time.

After the words are compiled, press "Start!" to begin the game.

The program will guess a word, and the user clicks the number of matching letters or "That's it!" if the secret word is guessed. This process repeats until the program guesses the right word, or it determines that it doesn't know the user's word.

About the program

At each step of the guessing game, the program selects the word that maximizes the entropy from the partitioning of words based on the number of matching letters. This is the next guess.
